## Specifications

- Distro: `Kali GNU/Linux Rolling x86_64`
- WM: `i3-gaps`
- Bar: `Polybar`
- Shell: `zsh 5.8`
- Text editor: `Vim`
- File manager: `thunar`
- Terminal: `kitty`
- Compositor: `compton` [ for background transparency ]
- Terminal apps: `neofetch`, `tty-clock`, `cmatrix`, `cava`
